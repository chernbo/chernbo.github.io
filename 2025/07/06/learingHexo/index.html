Hexo 学习 hexo butterfly 配置

hexo 博客基本布局如下：包括 顶部、侧边栏、文章列表、导航、底部；

hexo 文章类别

hexo new [layout] <name> #新建文章

布局类型 命令 保存位置 用途 特点
Post（文章） hexo new post <标题> 或 hexo new <标题> source/_posts/ 用于创建博客文章，最常用类型 - 显示在博客首页文章列表
- 包含日期、分类、标签等元数据
- 归档到时间线中
Page（页面） hexo new page <标题> source/<标题>/index.md 创建独立页面，如”关于我”、”联系方式”等 - 不显示在文章列表中
- 用于创建导航菜单独立页面
- 无日期、分类、标签等属性
Draft（草稿） hexo new draft <标题> source/_drafts/ 创建未完成或暂不发布的文章草稿 - 默认不显示在网站上
- 可用hexo publish <标题>发布
- 支持草稿预览（hexo s --draft）
- 适合存放未完成文章

Page 生成页面，还需要到 _config.butterfly.yml 文件中配置 menu
